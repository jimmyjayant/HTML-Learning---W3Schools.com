HTML also supports description lists.



HTML Description Lists


A Description list is a list of terms, with a description of each term.
The <dl> tag defines the description list, the <dt> tag defines the term (name), and the <dd> tag defines each term:-
Example:-

A Description List

Coffee
- Black Hot Drink
Milk
- White Cold Drink



Chapter Summary


  • Use the HTML <dl> element to define a description list
  • Use the HTML <dt> element to define a description term
  • Use the HTML <dd> element to define a term definition in the description list
